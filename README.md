# MathToolsGUI
MathToolsGUI is a Python-based graphical application that allows users to:  Calculate areas of various 2D shapes (rectangle, square, triangle, circle, parallelogram, trapezoid, ellipse, rhombus)  Calculate volumes of 3D shapes (cube, cuboid, sphere, cone, cylinder, pyramid, prism, hemisphere, ellipsoid)

Features:

Check Pythagorean triplets

Calculate complementary and supplementary angles

Switch easily between English and Persian interfaces

Easy-to-use GUI with tabs for Area, Volume, and Condition calculations

Error handling for invalid input
